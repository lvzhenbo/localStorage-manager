# localStorage管理器

用于添加、删除、修改localStorage和sessionStorage的简单插件。

感谢 **LocalStorage** (已被下架)，这个插件就是参照而做，只实现了基本的简单的功能，有更多需求欢迎PR。

## 项目设置

> 推荐使用pnpm

```sh
pnpm install
```

### 开发（编译和热重载）

```sh
pnpm dev
```

### 生产（类型检查、编译和最小化）

```sh
pnpm build
```

### 打包ZIP

```sh
pnpm zip
```
